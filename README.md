🔧 Project Goal:
Design a semi-automated tester for PCBs (starting with a simple LED driver or sensor board) that checks voltage levels, continuity, LED function, and outputs Pass/Fail.

📏 Design Rules I'm Using:
Trace width: 6–8 mil (signal), 20–30 mil (power)

Trace clearance: 6 mil

Component clearance: 15 mil

Board fab specs: 5 mil trace/clearance minimum
